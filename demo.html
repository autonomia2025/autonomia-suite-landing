Como iniciar la demo local

Corre el servidor Node y abre el sitio. El chat se conecta automaticamente si el backend esta disponible.

1. Instalar dependencias

Ejecuta npm install en el proyecto.

2. Iniciar servidor

Ejecuta npm start y revisa localhost:3000.

3. Explorar flujo

Abre el chat y observa la captura en tiempo real.
